Panduan Update Konten Website
  1. Siapkan Gambar: Pastikan gambar sudah dikompres (gunakan TinyPNG) dan namanya jelas (tanpa spasi).
  2. Upload Gambar: Buka repository GitHub Anda, masuk ke folder assets/images, lalu upload gambar baru.
  3. Edit Kode: Buka file index.html. Cari gambar yang ingin diganti dan ganti URL src-nya dengan nama file baru.
  4. Simpan & Commit: Simpan perubahan dan lakukan commit ke GitHub. Website akan otomatis terupdate.
